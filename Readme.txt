Объяснение решения задачи №4 из д/з урока №1 по теме "Ассимптотическая сложность и GIT":
Я решил данную задачу на палиндромы при помощи обращения к строке с помощью []. Таким образом я прочитал строку слева направо - [::1] и наоборот - [::-1] и сравнил полученные результаты между собой. Если строка при чтении слева направо совпадает со строкой при чтении справа налево, то такая строкаа является палиндромом и на экран выводится "True". Если же нет - то выводится "False".
